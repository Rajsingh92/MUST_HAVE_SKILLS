To follow along with the demos for this course you will need:

1. Putty to SSH to the Hadoop VM
http://www.putty.org/

2. Virtualbox or VMWare Player
https://www.virtualbox.org/wiki/Downloads

3. Hortonworks HDP Sandbox virtual machine image
We will go through setting this up in the first module but here's a link to the download if you want to start with that as you're watching the course material.

http://hortonworks.com/products/hortonworks-sandbox/
